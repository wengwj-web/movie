# node-mongodb-demo


# 安装部署
```
1. 安装 `Node.js[必须]` `MongoDB[必须]` `bower`
2. 启动 MongoDB
3. `npm install` 安装依赖包
4. `bower install bootstrap#3.0.0` bower 安装 bootstrap，同时jquery也一起安装了
5. 请根据需要修改启动文件app.js的MongoDB数据库连接信息
6. `node app.js`
7. visit `http://localhost:3000`
8. 先在 `http://localhost:3000/admin/movie` 录入电影信息 
```
